Our AST elements.

Variants

Number

A simple number object, this cannot be reduced further.

Add

An addition of two elements.

Multiply

A multiplication of two elements.

Boolean

A simple boolean object, this cannot be reduced further.

LessThan

A less-than relation check of two elements. Elements should reduce to a number to be comparable.

Variable

A variable, will be replaced by its value when reducing.

Assign

A variable assignment. Only completely reduced values are assigned. No type checks.

Sequence

A sequence of two elements. The first element is reduced completely before the second is touched.

IfElse

A if-else block. Condition needs to reduce to a Boolean. No type checking. If condition reduces to true, the consequence is used furhter, otherwise the alternative

While

A while loop. Runs until the condition reduces to false.

DoNothing

A simple no-op statement.
